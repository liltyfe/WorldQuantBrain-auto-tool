# 📓 量化进阶笔记：从“废案”到“高夏普”的 Alpha 因子优化手术刀

**📝 因子核心画像**
* **因子类型：** 价值反转复合因子 (Value + Reversal)
* **核心逻辑：** 寻找基本面极其低估（高自由现金流），且刚刚经历短线超跌的股票。
* **核心心法：** 绝不接飞刀，只在“卖方枯竭（缩量）”时入场捡漏。

## 💡 一、 原始底仓代码（待优化版）
```python
# 1. 估值基底：自由现金流收益率 (FCF Yield)
CBR_MV = (cashflow_op - capex) / cap

# 2. 核心信号：价值与短线收益率的回归打分 (这里用 10 天窗口更稳健)
alpha = winsorize(ts_regression(rank(CBR_MV), rank(returns), 10), std=3)

# 3. 市值中性化：消除小盘股偏见
signal = group_rank(alpha, bucket(rank(cap), range="0,1,0.1"))
```

---

## 🛠️ 二、 五步神级优化法则（提分核心）

### 法则 1：多因子线性叠加 (Alpha Blending)
> **操作：** `+ 0.2 * rank(-ts_delta(close, 3))`
* **代码解读：** 给基础的价值信号，加上 20% 权重的“近 3 天超跌”信号。
* **金融逻辑：** 慢变量（财务数据）结合快变量（量价动量）。不仅要买便宜的好公司，还要趁它最近刚挨过揍的时候买，提高短线反弹爆发力。

### 法则 2：市场环境过滤 (Regime Switching) ⭐️ 最核心
> **操作：** `s = trade_when(volume < adv20, signal, -1)`
* **代码解读：** 当且仅当今天成交量小于过去 20 天均值（缩量）时，采用上述做多信号；否则输出 -1（看空/观望）。
* **金融逻辑：** **缩量抄底，放量跑路。** 放量大跌意味着主力资金恐慌逃窜，极具危险性（接飞刀）；缩量大跌意味着抛压已经枯竭，微小的买盘就能撬动巨大涨幅。

### 法则 3：切断左侧尾部风险 (Tail Risk Control)
> **操作：** `ss = returns > -0.1 ? s : -1`
* **代码解读：** 如果今天单日跌幅超过 10%（例如 A 股跌停，或美股暴跌），无脑拉黑该信号，强行输出 -1。
* **金融逻辑：** 绝对的风控底线。单日极端暴跌通常预示着未知的“黑天鹅”事件（如财务造假、监管处罚），不要用头去碰基本面的雷。

### 法则 4：非线性信号拉伸 (Non-linear Mapping)
> **操作：** `reduce_signal = quantile(zscore(ss), driver = "cauchy")`
* **代码解读：** 先标准化，然后用柯西分布（Cauchy）重新映射分数。
* **金融逻辑：** 柯西分布属于“厚尾分布”。它会把平庸的信号压缩，把两极（极度看多 / 极度看空）的信号强行放大。这就等于告诉机器：“放弃那些没把握的交易，集中火力梭哈极其确定的机会！”

### 法则 5：换手率与成本控制 (Turnover Smoothing)
> **操作：** `ts_mean(reduce_signal, 15)`
* **代码解读：** 对最终生成的信号，取过去 15 天的滑动平均值。
* **金融逻辑：** 高频信号每天跳动会导致疯狂调仓，赚的钱全交了手续费。通过平滑处理，牺牲一点点敏捷度，换来换手率的骤降。**这是将纸面富贵（高 IS IR）转化为实盘真金白银（高净收益）的终极武器。**

完整代码:
```python
CBR_MV = (cashflow_op - capex)/cap;
alpha = winsorize(ts_regression(rank(CBR_MV),rank(returns),10),std=3);
signal = group_rank(alpha,bucket(rank(cap),range="0,1,0.1")) + 0.2*rank(-ts_delta(close, 3));
s = trade_when(volume<adv20,signal, -1);
ss = returns > -0.1 ? s : -1;
reduce_signal = quantile(zscore(ss), driver = "cauchy");
ts_mean(reduce_signal, 15)
```

## 📌 三、 核心思考总结 (Takeaways)

1.  **没有神奇公式，只有系统工程。** 顶级 Alpha 往往有一个平平无奇的基础逻辑，但拥有极其完善的防守机制。
2.  **把交易当成博弈。** 写公式时，脑子里要想的不是数据，而是市场里的“多头（推车的人）”和“空头（砸盘的人）”现在处于什么状态。
3.  **量化优化的通用清单：** 下次写完新因子，对照自查：做市值中性化了吗？加过滤条件防接飞刀了吗？切断单日暴跌雷区了吗？信号做平滑降换手了吗？
