# Alpha 批量生成工具使用说明

## 📋 快速开始

### 1. 运行工具
```bash
cd findalpha_mold/python_script
python generate_alphas_batch.py
```

### 2. 输入配置文件
```
请输入配置文件路径 (JSON格式):
  示例: ../alpha_configs/example_batch_config.json
配置文件路径: ../alpha_configs/example_batch_config.json
```

## 🔧 配置文件详解

### 基本结构
```json
{
  "name": "配置名称",
  "description": "配置描述",
  "output": {
    "file": "输出文件路径"
  },
  "templates": [
    // 模板配置...
  ],
  "data_files": {
    // 数据文件引用...
  }
}
```

### 字段说明

| 字段 | 是否必填 | 说明 |
|------|---------|------|
| `name` | ✅ 是 | 配置名称，用于显示 |
| `description` | ❌ 否 | 配置描述，用于说明用途 |
| `output` | ✅ 是 | 输出配置 |
| `output.file` | ✅ 是 | 输出文件的相对路径 |
| `templates` | ✅ 是 | 模板列表，可以有多个模板 |
| `data_files` | ❌ 否 | 数据文件引用映射 |

## 📝 模板配置

### 模板结构
```json
{
  "name": "模板名称",
  "template": "{group_op}({ts_op}({field}, {days}), {group})",
  "params": {
    "group_op": ["group_rank", "group_zscore"],
    "ts_op": ["ts_rank", "ts_zscore"],
    "field": "fundamental6_datafields",
    "days": [10, 20, 60],
    "group": ["market", "industry"]
  }
}
```

### 参数类型

| 参数类型 | 格式 | 示例 |
|---------|------|------|
| **固定列表** | 数组 | `["group_rank", "group_zscore"]` |
| **数据文件** | 字符串（以_datafields结尾） | `"fundamental6_datafields"` |
| **单个值** | 任意类型 | `252` |

## 💡 实际示例

### 示例1：简单单参数模板
```json
{
  "name": "简单排序",
  "template": "rank({field})",
  "params": {
    "field": "news12_MATRIX_datafields"
  }
}
```

### 示例2：双参数组合
```json
{
  "name": "TS排序组合",
  "template": "{ts_op}({field}, {days})",
  "params": {
    "ts_op": ["ts_rank", "ts_zscore", "ts_delta"],
    "field": "fundamental6_datafields",
    "days": [10, 20, 60, 120, 252]
  }
}
```

### 示例3：四参数组合（如图片所示）
```json
{
  "name": "四参数组合",
  "template": "{group_op}({ts_op}({field}, {days}), {group})",
  "params": {
    "group_op": ["group_rank", "group_zscore", "group_neutralize"],
    "ts_op": ["ts_rank", "ts_zscore", "ts_delta"],
    "field": "fundamental6_datafields",
    "days": [10, 20, 60, 120, 252],
    "group": ["market", "industry", "subindustry", "sector"]
  }
}
```

**这个配置会生成：3 × 3 × N × 5 × 4 = 180N 个Alpha表达式！**

## ⚠️ 注意事项

### 1. 路径说明
- 所有路径都是**相对于配置文件**的相对路径
- 不要使用绝对路径

### 2. 参数组合数量
```
参数组合数 = 每个参数选项数的乘积
```

例如：
- `group_op`: 3个选项
- `ts_op`: 3个选项  
- `field`: 100个字段
- `days`: 5个选项
- `group`: 4个选项

**总数 = 3 × 3 × 100 × 5 × 4 = 18,000 个Alpha！**

### 3. 性能考虑
- 参数组合太多会导致生成很慢
- 建议分批生成，或者减少参数选项

## 🚀 常见问题

### Q: 如何添加新的数据文件？
A: 在 `data_files` 中添加：
```json
"data_files": {
  "my_data": "../Data/my_data.csv"
}
```

### Q: 如何使用同一个数据文件多次？
A: 可以多次引用同一个数据文件键

### Q: 单个模板和批量工具的区别？
| 特性 | generate_alphas.py | generate_alphas_batch.py |
|------|-------------------|--------------------------|
| 模板数量 | 1个 | 多个 |
| 参数组合 | 不支持 | 支持 |
| 配置文件 | 不需要 | 需要 |
